## 📋 Project Overview

This project showcases an interactive Tableau dashboard that analyzes and visualizes global library data, offering valuable insights into the distribution and performance of library services around the world. The dashboard serves as a powerful tool for understanding disparities in library access, resources, and funding across different geographical regions.

## ✨ Key Features

### Interactive Visualizations
- **Regional Library Distribution**: Visual representation of library density across different world regions
- **Country-Level Analysis**: Detailed breakdown of library statistics by individual countries
- **Comparative Metrics**: Side-by-side comparison of key library indicators
- **Dynamic Filtering**: Interactive filters for regions and specific indicators

### Key Metrics Analyzed
- 📍 **Total Libraries by Country & Region**: Geographic distribution of library facilities
- 📖 **Volumes Held**: Collection sizes and availability of reading materials
- 💰 **Library Expenditures**: Investment and funding patterns in library services
- 👥 **Population Served**: Coverage and accessibility of library services to communities
- 🌍 **Regional Comparisons**: Cross-regional analysis of library performance

### Dashboard Capabilities
- **Drill-Down Analysis**: Click-through functionality for deeper data exploration
- **Outlier Identification**: Highlighting of exceptional cases and disparities
- **Data Storytelling**: Clean, intuitive design that guides users through insights
- **Interactive Filters**: Dynamic selection of regions and performance indicators

## 🚀 Getting Started

### Prerequisites
- Tableau Desktop (recommended) or Tableau Public access
- Basic understanding of data visualization concepts

### Installation & Usage

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Rajawialqhtani/tableau-global-library-dashboard.git
   cd tableau-global-library-dashboard
   ```

2. **Open the Dashboard**
   - Locate and open the `.twbx` file in Tableau Desktop
   - Alternatively, view the dashboard on Tableau Public (if published)

3. **Explore the Data**
   - Use the interactive filters to explore different regions
   - Click on countries or regions for detailed analysis
   - Examine the various metrics and their relationships
